vibe coded live latex editor web demo using katex, inspired by the one on their website [katex.org]

### features
- autosave
- backups every 3 minutes
- dark/light mode
- vim keybindings
- edit multiple files

### how to use

```bash
git clone https://github.com/rohans0/live-latex-editor
cd live-latex-editor
node install
node server.js
```
